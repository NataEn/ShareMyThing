# Milestones

## What is ShareHub?

A web application where users can share things that are not in use, e.g. furniture, home appliances, books etc.

## Scope and Foundations

- **Front-End** &ndash; React
- **Back-End** &ndash; nodeJs
- **Database** &ndash; Mongo
